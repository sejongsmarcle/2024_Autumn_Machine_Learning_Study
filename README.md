```python
- 출석 | ~7시 5분
- 지각 | 7시 5분~15분
- 결석 | 7시 15분~
```

### 과제제출기준
```python
- 제출 | 스터디 당일 12시까지
- 지각 | 스터디 당일 12시~7시까지
- 미제출 | 스터디 시작 이후
```

### 경고
```python
- 지각(과제 지각) 2회 혹은 결석(미제출) 1회 시 스터디 경고를 부여한다.

```
